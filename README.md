## 🌟 Project Description

Grüner Mars is a research synthesis and interactive roadmap for building sustainable Martian habitats using biology-first approaches:

- **Pioneer Microbes**: Perchlorate-reducing bacteria (PRB) and Chroococcidiopsis for soil detoxification and O₂ production.
- **Mycotecture**: Radiotrophic fungi (Cladosporium sphaerospermum) for radiation shielding and construction materials (NASA NIAC heritage).
- **Closed-Loop Systems**: Multi-loop aquaponics (Goddek model) producing food, fish, and Omega-3 from CO₂.
- **3D Fractal Hügelbeete**: Volume-efficient, mycorrhiza-enhanced raised beds for maximum yield in habitat constraints.
- **Hybrid Resilience**: 60-70% Earth regenerative bio + 30-40% space-ag/tech allocation for long-term viability.

The app includes interactive Mermaid diagrams, Chart.js visualizations, timeline filters, X-share buttons, and dedicated subpages.
