# Grüner Mars

**Hybrid Regenerative Biology & Space Agriculture Feasibility 2026**

A professional static web app exploring biological feasibility for Mars colonization, focusing on perchlorate-tolerant microbes, 3D fractal Hügelbeete, radiation shielding via mycotecture, closed-loop aquaponics, and hybrid resilience models (60-70% Earth Bio + 30-40% Space-Ag).

- Live demo: https://rosary-mom.github.io/gruener-mars (GitHub Pages)
- Part of rosary-mom projects and #GrünerMars campaign
- Integrates with DRI Slot 30, ESG-Spiral-Tokens, rosary.health, and Magnifica Humanitas

## Features
- Enhanced roadmap with interactive Mermaid diagrams and Chart.js
- Dark/light mode toggle
- Interactive timeline and filters
- Share on X buttons with #GrünerMars
- Subpages: Roadmap, Deep-Dive, Investor (DRI Slot 30), Blog
- Responsive, self-contained static site (Tailwind, Mermaid, Chart.js via CDN)

## Structure
- `index.html` - Main landing + enhanced roadmap
- `assets/` - Additional CSS/JS if needed
- `blog/` - Blog posts
- `roadmap/` - Detailed roadmap subpage
- `README.md`, `LICENSE`

## Local Development
Open `index.html` in a browser. For full site, serve with any static server:
```bash
python -m http.server 8000
```
